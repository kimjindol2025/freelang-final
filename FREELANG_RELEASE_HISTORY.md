## 🔧 기술 진화

### 컴파일러 아키텍처 변화

**초기 (v2.0.0)**
```
FreeLang → Lexer → Parser → IR → VM
```

**중간 (v2.1.0)**
```
FreeLang → Lexer → Parser → Type Checker → Optimizer → IR → VM
              ↑                                              ↑
              └─────── Module System ──────────────────────┘
```

**현재 (v2.2.0)**
```
FreeLang → Lexer → Parser → Type Checker → Optimizer → IR ──┐
              ↑                                              ↓
              └─ Module System + KPM ─ GCC/NASM/x86-64 ← Codegen
                        ↑
                   Self-Hosting
                   (FreeLang.fl)
```

| 버전 | 파싱 | 타입 체크 | 최적화 | 컴파일 | 총합 |

| v2.0.0 | 10ms | 5ms | - | 15ms | **30ms** |
